Capa 1: Física

Hardware de Servidores y Dispositivos de Usuarios

Incluye servidores web donde reside el sitio y dispositivos (computadoras, teléfonos móviles) utilizados por los usuarios para acceder al sitio.

Capa 2: Enlace de Datos

Red Local y Conexión a Internet

Routers, switches, y otros equipos que facilitan la conexión a internet tanto para el servidor como para los usuarios finales.

Capa 3: Red

Protocolos de Internet

Utilización de TCP/IP para la transferencia de datos entre el servidor y los clientes (navegadores de los usuarios).

Capa 4: Transporte

Protocolo de Transporte

Uso de TCP para control de flujo, segmentación de datos y manejo de errores; HTTPS para asegurar la comunicación entre cliente y servidor.

Capa 5: Sesión

Gestión de Sesiones Web

Manejo de sesiones de usuario en el sitio web, aunque no se requiere registro. Esto incluye la interacción con el chatbot y el mantenimiento del estado durante la navegación.

Capa 6: Presentación

HTML, CSS, y JavaScript

Estos lenguajes se utilizan para estructurar el contenido, definir la presentación y la interactividad en el navegador del usuario. Aquí también se incluye la transformación de datos (por ejemplo, JSON) utilizados para la comunicación entre el cliente y el servidor.

Capa 7: Aplicación

Descripción breve...

Frontend: El navegador web del usuario, interpretando HTML, CSS y JavaScript. Backend: Servidores web que ejecutan código en Python (y posiblemente Java), gestionando la lógica del negocio, la base de datos y la integración del chatbot. APIs y Servicios Web: Interfaces utilizadas para la comunicación entre el chatbot, la base de datos del servidor y posiblemente servicios externos.
